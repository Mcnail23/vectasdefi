# VectasDeFi 🚀

An asynchronous, real-time on-chain analytics and security monitoring framework designed to track decentralized finance (DeFi) metrics, smart contract activity, and mempool anomalies.

## 🔍 Features
* **Real-time On-Chain Monitoring:** Tracks liquidity changes, large token transfers, and smart contract deployments.
* **Threat & Exploit Detection:** Scans for signature anomalies and potential DeFi vectors.
* **High-Performance Architecture:** Asynchronous engine built for speed and efficiency.
